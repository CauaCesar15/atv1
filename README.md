1. Objetivo Geral 

O objetivo deste software é desenvolver uma calculadora web capaz de realizar operações matemáticas básicas com valores numéricos. 

A aplicação será voltada para facilitar o uso do usuário, permitindo realizar cálculos de forma rápida e intuitiva através de uma interface semelhante a uma calculadora convencional. 

A calculadora deverá possuir as seguintes funcionalidades: 

Soma 

Subtração 

Multiplicação 

Divisão 

O design da aplicação deverá se aproximar de uma calculadora real, tanto em aparência quanto em funcionalidade, proporcionando uma experiência simples e familiar ao usuário. 

 

2. Regras de Negócio 

RN-01 – Interface simples e intuitiva 

A calculadora deverá possuir uma interface clara e fácil de utilizar, permitindo que o usuário realize cálculos de forma natural, semelhante ao uso de uma calculadora tradicional. 

 

RN-02 – Limitação de botões numéricos 

A calculadora deverá possuir botões numéricos de 0 a 9, além dos botões correspondentes às quatro operações matemáticas básicas: 

Soma (+) 

Subtração (−) 

Multiplicação (×) 

Divisão (÷)
